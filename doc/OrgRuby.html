require_all_libs_relative_to( fname, dir = nil ) click to toggle source

Utility method used to require all files ending in .rb that lie in the directory below this file that has the same name as the filename passed in. Optionally, a specific directory name can be passed in such that the filename does not have to be equivalent to the directory. 

    # File lib/org-ruby.rb, line 38
38:   def self.require_all_libs_relative_to( fname, dir = nil )
39:     dir ||= ::File.basename(fname, '.*')
40:     search_me = ::File.expand_path(
41:         ::File.join(::File.dirname(fname), dir, '**', '*.rb'))
42: 
43:     Dir.glob(search_me).sort.each {|rb| require rb}
44:   end
